Prototype Point of Sale System

This system was developed over the course of a couple of days by Finlay Maroney. It is designed to avoid some of the issues encountered with existing solutions and is designed to be (hopefully) fairly secure. It uses local Unix sockets for authentication (can be used over SSH as well for remote terminals) which minimizes the access of any terminal. This site is not meant to be a finished system and a proper POS system would likely be a local client for this (it has an easy to use JSON API that allows integration fairly easily with local clients) as it allows for interactions with hardware such as pin pads and cash drawers more easily. I am also not huge on front end development so please excuse the lack of styling.
